# Site Web - Le Roche Imprimé

## Description
Site web professionnel pour "Le Roche Imprimé", une entreprise spécialisée dans l'impression professionnelle. Le site présente tous les services d'impression sur une seule page avec un design moderne et responsive.

## Fonctionnalités

### Sections du Site
- **Header avec navigation fixe** : Menu de navigation smooth scroll
- **Section Héro** : Présentation de l'entreprise avec call-to-action
- **À Propos** : Histoire de l'entreprise et valeurs
- **Services d'Impression** :
  - Impression Offset
  - Impression Numérique
  - Impression Grand Format
  - Cartes de Visite
  - Brochures & Catalogues
  - Flyers & Dépliants
- **Portfolio** : Galerie d'exemples de réalisations avec lightbox
- **Témoignages** : Carousel de témoignages clients
- **Contact** : Coordonnées complètes et formulaire de devis
- **Footer** : Informations supplémentaires et liens

### Fonctionnalités Interactives
- Navigation smooth scroll
- Menu mobile responsive avec hamburger
- Carousel automatique des témoignages
- Lightbox pour agrandir les images du portfolio
- Formulaire de contact avec validation
- Liens directs vers téléphone et WhatsApp
- Bouton de retour en haut de page
- Animations au scroll

## Responsive Design
Le site est entièrement responsive et s'adapte à tous les écrans :
- Desktop (1200px+)
- Tablette (768px - 1199px)
- Mobile (320px - 767px)

## Installation et Utilisation
1. Télécharger tous les fichiers
2. Ouvrir `index.html` dans un navigateur web
3. Le site fonctionne directement sans serveur

## Optimisations
- Images optimisées pour le web
- Code CSS et JavaScript minifiés
- Animations fluides avec transitions CSS
- Lazy loading des images
- Performance optimisée

## Maintenance
Pour modifier le contenu :
1. **Textes** : Éditer directement dans `index.html`
2. **Styles** : Modifier `css/style.css`
3. **Interactions** : Ajuster `js/script.js`
4. **Images** : Remplacer dans le dossier `images/`
